# 系统
1. Linux>MacOS>Win
2. 可选[[QubesOS]]或[[Pop! OS]]，用[[Tails]]辅助

# 初始化
1. 全盘加密
2. 关闭蓝牙和ConnectivityChecking
3. 禁用FileHistory&Trash的所有选项
4. 使用Pop! Shop和apt更新所有应用

# 网络
1. 使用[[Next DNS]]，并且进行隐私导向配置，然后彻底关闭日志功能
2. 别用[[ADGuard]]
3. 使用值得信任的VPN
4. 考虑使用Tor和i2p
